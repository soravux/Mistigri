# Mistigri
Multiplayer Web-based Game Engine

General Setup
=============

Install Node.js, then:
```
npm install -g bower
npm install -g grunt
```

Server Setup
============

```
cd server
npm install
node .
```

Client Setup
============

```
cd client
npm install
bower install
grunt build
```
Then, go to `client/index.html` with a browser
