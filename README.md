# Mistigri
Multiplayer Web-based Game Engine

General Setup
-------------

Install Python 3.4 and Node.js, then:
```
npm install -g bower
npm install -g grunt
```

Server Setup
------------

```
cd server
npm install
node .
```

Client Setup
------------

```
cd client
npm install
bower install
grunt build
python -m http.server
```
Then, go to `http://127.0.0.1:8000` with a browser
